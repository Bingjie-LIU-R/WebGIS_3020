本网页讲述的故事以读书时间节点为主，讲述部分成长中的故事。
第一部分Childhood，主要为与爷爷奶奶生活时期，在宜都市全福河村，网页中用Leaflet.ChineseTmsProviders加载底图显示矢量地图与Google影像图，点击右上角可进行地图切换，放大影像图POP窗口定位家的位置；
第二、三部分主要讲到了读书的年纪，跟随父母在磨市镇定居，上小学与初中，用高德地图路线规划显示两地之隔，同时以家为参照，展示两所学校相对家的位置；
第四部分主要展示高中校园与“三点一线”的生活，通过Google高分遥感影像与矢量对象区分各个区域，并讲述高中平淡无奇生活中，自己的一个勇敢尝试；
第五部分调用百度地图API，添加全景展示湖北大学，并添加大学健身、实习、与兴趣培养分析；
【HTML布局以Google浏览器为准，最后全景展示有对Flash Player要求，或者采用火狐浏览器打开】
